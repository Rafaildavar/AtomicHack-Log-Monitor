AtomicHack Log Monitor - интеллектуальная система анализа логов с ML

Решение автоматически анализирует логи инфраструктуры, выявляет аномалии (WARNING) через семантический поиск с помощью Sentence Transformers (модель all-MiniLM-L6-v2) и находит связанные корневые проблемы (ERROR) с точной локализацией в файлах.

Ключевые возможности:
• Telegram Bot интерфейс для загрузки логов (до 20MB)
• ML-анализ с порогом уверенности 0.7 (устойчивость к вариациям текста)
• Встроенный словарь: 501 аномалия, 50 типов проблем
• Поддержка кастомных словарей и множественных сценариев
• Структурированные Excel отчеты с полной трассировкой
• Производительность: 30 сек на 1.5 млн строк логов

Технологии: Python 3.13, aiogram 3.22, sentence-transformers 5.1, PyTorch 2.8, pandas 2.3, openpyxl 3.1
